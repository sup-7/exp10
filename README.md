Algorithm

Create Dynamic Web Project.
Start MySQL Server.
Create database and users table.
Create HTML registration form. --> index.html in webapp
Create Servlet class. --> RegisterServlet.java in src/main/java
Read form parameters in servlet.
Load JDBC driver.
Establish database connection.
Insert data into database table.
Display registration message.

Configure web.xml --> web-app in web.xml
Run project on Tomcat server.
